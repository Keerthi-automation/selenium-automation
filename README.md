# Selenium Automation Cucumber Framework
 
This project is a Selenium automation testing framework built using:
 
- Java
- Selenium WebDriver
- Cucumber BDD
- Maven
- Extent Reports
 
Framework Features:
- Page Object Model
- Cucumber Feature Files
- Step Definitions
- Extent Reporting
- Maven Dependency Management
 
Project Structure:
src/test/java
    pages
    steps
    runners
 
src/test/resources
    features
 
How to Run:
Run the Runner class or execute:
mvn test
